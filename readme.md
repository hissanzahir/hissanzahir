## Featured Projects

### F1GPT

**Tech:** `Next.js` `LangChain.js` `OpenAI` `Astra DB`

A Retrieval-Augmented Generation chatbot that uses semantic search and vector embeddings to answer complex Formula 1 questions.

### Diabetic Retinopathy Detection

**Tech:** `PyTorch` `EfficientNet-B4` `FastAPI` `React Native` `Supabase`

An end-to-end AI system for diabetic retinopathy classification. It features Grad-CAM explainability for clinical transparency and includes a companion React Native mobile app for users.
